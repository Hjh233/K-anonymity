You can find adult.data via https://archive.ics.uci.edu/ml/datasets/adult 
Attributes is composed of 'age', 'work_class', 'final_weight', 'education',
            'education_num', 'marital_status', 'occupation', 'relationship',
            'race', 'sex', 'capital_gain', 'capital_loss', 'hours_per_week',
            'native_country', 'class'
adult_*.txt: You can find two strings each column with separator being ','. 
The first donates the child node and the second representing the parent node.
